Library Borrowing System

How to use:
1. Copy the folder library_borrowing into your XAMPP htdocs folder.
2. Start Apache and MySQL in XAMPP.
3. Open phpMyAdmin.
4. Import database.sql.
5. Open this URL in your browser:
   http://localhost/library_borrowing/register.php
6. Register a new member, login, and test the borrowing system.

Files included:
- database.sql
- db_connect.php
- register.php
- login.php
- home.php
- update.php
- delete.php
- logout.php
